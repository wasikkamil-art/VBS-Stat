# Email do widziszwszystko (krótka wersja)

## Temat
```
Integracja z Harmonogramami DDD — czy można pobierać pliki przez API lub email?
```

## Treść

---

Dzień dobry,

korzystamy z Państwa monitoringu dla 6 pojazdów (login **vbs**), mamy pakiet
**Premium z Harmonogramami DDD**. Budujemy własny system zarządzania flotą
i chcielibyśmy pobierać do niego pliki .ddd które pobieracie z naszych tachografów.

Trzy pytania:

**1. Czy można ustawić automatyczne wysyłanie plików DDD na nasz email**
po każdym pobraniu przez harmonogram? (widziałem na Państwa YouTube film
o tej funkcji — chciałbym ją aktywować)

**2. Czy jest API do pobierania plików DDD** — analogiczne do Atlas API
które używamy do pozycji GPS? Jeśli tak, proszę o dokumentację.

**3. Czy nasze urządzenia GPS w pojazdach obsługują zdalny odczyt DDD**,
czy potrzebujemy upgrade'u?

Chcemy to zrobić zgodnie z Państwa regulaminem — dlatego pytamy wprost
zamiast kombinować po swojemu.
